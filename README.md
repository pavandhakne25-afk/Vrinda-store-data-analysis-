# 📊 Vrinda Store Data Analysis

This project focuses on analyzing sales data of Vrinda Store using Microsoft Excel.

## 🔹 Project Overview
- Analyzed customer data, orders, and sales
- Used filters, sorting, and basic Excel operations
- Explored trends based on gender, age, and region

## 🔹 Tools Used
- Microsoft Excel

## 🔹 Key Insights
- Majority customers are Women
- Most orders are from specific states
- Sales vary based on category and channel

## 🔹 File Included
- Vrinda_Store_Analysis.xlsx

## 🚀 Purpose
This project is created for learning and showcasing Excel data analysis skills.
